# Highest-Grossing Films of All Time

An interactive web dashboard visualising the 50 highest-grossing films ever, built for the **Data Wrangling and Visualization** course.

**Live demo → [alyaalyo.github.io](https://alyaalyo.github.io)**

---

## Project overview

| Step | What happens |
|------|-------------|
| **Scraping** | `scraper.ipynb` fetches the Wikipedia page [List of highest-grossing films](https://en.wikipedia.org/wiki/List_of_highest-grossing_films) using `requests` + `BeautifulSoup` |
| **Enrichment** | Each film's individual Wikipedia page is visited to extract director(s) and country of origin from the infobox |
| **Storage** | Cleaned data is written to a local `films.db` SQLite database |
| **Export** | The database is serialised to `films.json` so the static web page can consume it without a backend |
| **Frontend** | `index.html` / `script.js` / `style.css` render the data with Chart.js, search, filters, and a grid/table toggle |
| **Deployment** | The frontend is hosted on GitHub Pages |

## Database schema

```sql
CREATE TABLE films (
    id             INTEGER PRIMARY KEY AUTOINCREMENT,
    rank           TEXT,
    title          TEXT NOT NULL,
    release_year   INTEGER,
    director       TEXT,
    box_office     TEXT,      -- formatted string, e.g. "$2,923,710,708"
    box_office_usd REAL,      -- numeric value in USD for sorting
    country        TEXT
);
```

## How to run locally

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

Requires **Python 3.9+**.

### 2. Run the notebook

```bash
jupyter notebook scraper.ipynb
```

Run all cells. The notebook will create `films.db` and overwrite `films.json`.

### 3. Serve the frontend

GitHub Pages serves static files, but browsers block `fetch()` on `file://` URLs.  
Use any local HTTP server, for example:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

## Frontend features

- **Bar chart** — top 15 films by box office revenue (Chart.js)
- **Search** — filter by title or director in real time
- **Country filter** — dropdown built from the actual data
- **Year range** — numeric inputs to narrow by release decade
- **Sort** — click any column header in table view
- **Grid / Table toggle** — two display modes
